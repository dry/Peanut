h2. What is it?

Peanut is a lightweight file-based content management system. It is ideal for brochure-style sites or application documentation. It has no dependencies and very few requirements:

* PHP 5
* A server rewriting module
--

body:
h2. Features

* "Zero configuration required":/documentation/install.html
* "Multiple layouts":/documentation/layouts.html
* Layouts can be defined on a per-page basis
* Unlimited custom fields
* Plugin-based text parsing (Textile is included)
* URLs defined by file and folder structure
* Configurable file extensions
